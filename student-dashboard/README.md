## Student Dashboard UI

## Overview
This is a responsive web dashboard designed to help students track their daily schedule, upcoming assignments, and study streaks. It's a simple, intuitive interface that supports student organization and motivation - inspired by Clever's mission to connect every student to a world of learning.

## Features
-Daily schedule display with static class times
-Upcoming assignments list
-Study streak tracker with local storage persistence
-Responsive layout for desktop and mobile
-Clean, accessible design using semantic HTML and CSS

## Technologies Used
-HTML5
-CSS3
-JavaScript
-LocalStorage API

##  Why It Matters
This project demonstrates my ability to build user-centric interfaces with clear utility and empathetic design. It reflects Clever's values by supporting student agency and engagement through simple, effective tools.

## How to Run
Open 'index.html' in any modern browser. No installation required.

## Reflection
I built this project to practice DOM manipulation, layout design, and state persistence. If expanded, I'd add dynamic assignment loading, calender integration, and personalized streak goals to deepend student engagement.
